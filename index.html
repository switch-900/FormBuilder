The Problem

AI conversations can be inefficient:

  • You ask a question, AI asks 3 follow-up questions
  • You answer those, AI asks 3 more
  • Multiple rounds of back-and-forth before real work begins
  • Information gets scattered across messages
  • Hard to track what you've already provided

The Solution: Structured Forms

Instead of playing 20 questions, the AI gives you a JSON schema with all the questions upfront. You fill out a clean form, export your answers as JSON, and paste it back. One round trip instead of five.

How It Works

1

AI Provides Schema

Your AI assistant (ChatGPT, Claude, etc.) gives you a JSON schema with all the questions it needs answered.

2

You Fill the Form

Paste the schema into the Form Builder. It generates a clean form with text inputs, dropdowns, checkboxes, and more.

3

Export Answers

Fill out the form at your own pace. Click "Copy answers JSON" to get a clean, structured JSON with all your responses.

4

Paste Back to AI

Send the answers JSON back to your AI. It now has everything it needs to continue without asking more questions.

Features

7 Field Types

Text, textarea, numbers, checkboxes, radio buttons, select dropdowns, and multi-select checkboxes.

100% Local

Everything runs in your browser. No data sent to servers. Your information stays private.

Clean Export

One-click copy to clipboard. Answers JSON is ready to paste directly into your AI conversation.

AI Prompt Generator

Example Schema

Here's what a simple schema looks like:

{
  "version": "1.0",
  "title": "Website Design Preferences",
  "description": "Quick questions to lock in the design direction",
  "fields": [
    {
      "id": "color_scheme",
      "label": "Choose a color scheme",
      "type": "select",
      "required": true,
      "options": [
        { "value": "dark", "label": "Dark mode (like this site)" },
        { "value": "light", "label": "Light and minimal" },
        { "value": "colorful", "label": "Vibrant and colorful" }
      ]
    },
    {
      "id": "must_have",
      "label": "Must-have features",
      "type": "checkboxes",
      "options": [
        { "value": "mobile", "label": "Mobile responsive" },
        { "value": "animations", "label": "Smooth animations" },
        { "value": "forms", "label": "Contact forms" }
      ]
    },
    {
      "id": "inspiration",
      "label": "Link to sites you like (optional)",
      "type": "textarea",
      "placeholder": "Paste URLs or describe what you like..."
    }
  ]
}

Getting Started

For AI Users:

  1. Copy the "Schema + AI Prompt" from the form builder
  2. Paste it into your AI conversation (ChatGPT, Claude, etc.)
  3. The AI will understand the format and provide schemas when needed

For AI Assistants:

When you need structured information from a user:

  1. Create a JSON schema following the format shown in examples
  2. Provide it to the user with clear instructions
  3. Wait for their answers JSON response
  4. Use the structured data to continue efficiently

Benefits
